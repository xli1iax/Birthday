
# Birthday Website 🎂

An interactive birthday greeting webpage created as a small festive project with animations and styled design.

## Description

This project is a simple web page dedicated to celebrating a birthday.  
It includes visual elements, animations, and a styled layout to create a festive atmosphere.

The website is deployed using GitHub Pages and available at:

https://xli1iax.github.io/Birthday/

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages

## Project Structure

```text
Birthday/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── content/
    └── images / media files
````

## Features

* festive landing page
* responsive layout
* animations and decorative elements
* external styles and scripts
* deployed via GitHub Pages

## How to Run

You can open the project directly via:

[https://xli1iax.github.io/Birthday/](https://xli1iax.github.io/Birthday/)

Or run it locally:

1. Clone or download the repository
2. Open `index.html` in your browser

## GitHub Pages

The project is deployed from the `main` branch using the `/root` folder.
